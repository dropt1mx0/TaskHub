# TaskHub — Карта анимированных стикеров

Все стикеры лежат в `/webapp/static/stickers/`.
Формат: **Lottie JSON** (`.json`) — это TGS-стикеры из Telegram, распакованные из gzip.

---

## Как получить Lottie JSON из Telegram-стикера

### Способ 1: Через бота @Stickers
1. Отправь стикер боту `@taborigbot` или `@cactusrobot`
2. Он вернёт файл `.tgs`
3. `.tgs` = gzip Lottie JSON. Распакуй: `mv sticker.tgs sticker.json.gz && gunzip sticker.json.gz`
4. Получишь `sticker.json` — положи в папку `stickers/` с нужным именем

### Способ 2: Через LottieFiles
1. Зайди на https://lottiefiles.com
2. Найди нужную анимацию
3. Скачай как Lottie JSON
4. Переименуй и положи в папку

### Способ 3: Через Telegram Desktop
1. Нажми ПКМ на стикере → "Сохранить как..."
2. Сохранится `.tgs` файл
3. Распакуй: `gunzip -S .tgs sticker.tgs` → получишь Lottie JSON

## Как добавить

1. Получи Lottie JSON любым способом выше
2. Переименуй по таблице (например `splash.json`)
3. Положи в `/webapp/static/stickers/`
4. Всё! Приложение автоматически подхватит файл вместо CDN-заглушки

## Техническая заметка
Каждый слот — `<div class="sticker-slot">` в HTML.
JS при загрузке проверяет наличие локального файла (`fetch HEAD`).
Если файл есть → `<lottie-player src="/static/stickers/xxx.json">`.
Если нет → `<lottie-player src="https://assets2.lottiefiles.com/...">` (заглушка).
